== Description ==

**Whatsapp Share Button - Share on whatsapp**

Whatsapp Share is the premium social share plugin that lets you create whatsapp share button for your blog.Easy to configure just place shortcode where you want to add whatsapp share button. 



### Features

* Whatsapp Share Button 
* Three type social share button
* Sortcode [whatsapp_share]
* Supported Argument
    1) "button_text" Link Text
    2) "number" Number to send message
    3) "message_type"  possible value only_number,direct_message,only_message 
    4) "message" Message to send (keep blank for current page url)
    5) "id" Link Id
    6) "class" Link Class
    7) "new_tab" possible value "true" or "false"

* For Suggestion or Query Contact at <a href="http://www.wapinfosolutions.com?ref=wordpress" target="_blank">wapinfosolutions</a>

* Stuck at any point <a href="https://api.whatsapp.com/send?phone=9016512020&text=I%20need%20support&l=en">contact us </a>

### Sample Shortcode
 [whatsapp_share button_text="Contact Us" number="9016512020" id="whatsapplink" class="link1 button responsive link" new_tab="true" message_type="direct_message"]

  

== Installation ==
How to install the Whatsapp Share 

Please follow the steps for your Whatsapp Share  installation below: 

= VIA WordPress =
1. After your download log into your WordPress admin interface
2. After login click on *Plugins* from the left hand menu
3. Click on *Add new*
4. Click on *Upload*
5. Browse to the directory you downloaded the plugin to and click *Install Now*
6. After Wordpress has finished unpacking the file click on *Activate Plugin*
7. Done. Start placing shortcode!

= VIA FTP =
1. After your download unzip whatsapp_share from your download .zip
2. Open your FTP client
3. Browse to /your_wp_instalation/wp-content/plugins/ on your hosting server
4. Uplaod whatsapp share into this directory
5. Go to your wordpress admin panel
6. Go to plugins and activate whatsapp share
7. Done. Start placing shortcode!
